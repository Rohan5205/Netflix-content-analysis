# Netflix Content Analysis

## Tools Used
Python, Pandas, Matplotlib, Power BI

## Project Overview
Analysis of Netflix movies and TV shows to identify trends in genres, countries, and release patterns.

## Key Insights
- Most content released after 2015
- Drama and Comedy dominate Netflix
- USA and India produce the highest number of titles
